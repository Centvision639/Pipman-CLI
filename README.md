## 🧭 What Pipman-CLI does

Pipman-CLI helps you manage Python packages from a simple terminal screen. It scans your Python setup, checks what is installed, finds package updates, and shows results in a clear format.

Use it when you want to:
- see which Python packages are installed
- find packages that need updates
- update packages with less typing
- keep your Python tools in one place
- get clear output without sorting through long command lines

## 🛠️ Troubleshooting on Windows

If the tool does not start, check these items:
- Python is installed
- You opened the right folder
- You ran the correct file
- The file was not blocked by Windows
- Your internet connection works for package checks
- You opened Command Prompt in the same folder as the tool

If Windows says the file is blocked:
1. Right-click the file
2. Choose Properties
3. Look for an Unblock box
4. Check it
5. Select OK

If a command fails, type it again and check for spelling errors.
